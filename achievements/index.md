# 成果展示

本页面用于整理我目前取得的科研成果、项目经历和阶段性工作。

我的本科阶段研究主要集中在 **Web API 推荐** 方向，关注如何根据 Mashup 或用户需求，为开发者推荐合适的 Web API 服务或服务组合。

---

## 1. 科研成果

### [TSSGCF: Textual Similarity-Supervised Graph Collaborative Filtering for Web API Recommendation](tssgcf.html)

- 会议：ICSOC 2025
- 类别：CCF B 类会议
- 作者身份：第二作者
- 方向：Web API Recommendation / Graph Collaborative Filtering / Textual Similarity

该工作主要研究如何利用 Mashup 与 Web API 的文本语义相似性监督图协同过滤过程，从而提升 Web API 推荐效果。

---

### [Multi-Modal Complementary Web API Recommendation with Attentive Context Modeling](mmc-webapi.html)

- 会议：CSCWD 2026
- 类别：CCF C 类会议
- 作者身份：第一作者
- 方向：Multi-Modal Web API Recommendation / Complementary Recommendation / Attention Mechanism

该工作主要研究互补式 Web API 推荐任务，关注如何根据已有 Web API 组合，为开发者推荐具有互补价值的新 API。
