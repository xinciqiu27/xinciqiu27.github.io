# TSSGCF: Textual Similarity-Supervised Graph Collaborative Filtering for Web API Recommendation

## 基本信息

- 论文题目：TSSGCF: Textual Similarity-Supervised Graph Collaborative Filtering for Web API Recommendation
- 会议：The 23rd International Conference on Service-Oriented Computing, ICSOC 2025
- 类别：CCF B 类会议
- 作者身份：第二作者
- 研究方向：Web API Recommendation, Graph Collaborative Filtering, Textual Similarity

---

## 摘要信息

这篇论文聚焦于 **Web API 推荐中的图协同过滤问题**。论文指出，现有 GNN 类方法虽然能够在 Mashup-API 交互图上捕捉高阶依赖关系，但也存在三个明显局限：一是多层图卷积容易引入噪声并造成过平滑，导致不同节点的表示越来越难区分；二是交互数据本身较为稀疏，长尾 API 和冷启动 API 很难学到可靠表示；三是许多方法过于依赖交互图结构，没有充分利用 Mashup 和 Web API 的文本描述信息。

为此，论文提出 **TSSGCF**。该方法先利用文本语义构建去噪的相似图，再结合 LightGCN 学习图结构表示，同时引入**文本相似度监督机制**，让节点表示不仅保留协同过滤信号，还受到文本语义的直接引导，从而提升表示的判别性。实验结果表明，TSSGCF 在真实数据集上相较最强基线取得了 **Recall +9.44%**、**NDCG +20.14%**、**Precision +8.38%** 和 **F1 +8.56%** 的提升；消融实验进一步说明，去噪相似图、文本语义表示和文本相似度监督三个模块都对性能提升有直接贡献。

---

## 研究背景

Web API 推荐任务旨在根据 Mashup 的功能需求或历史交互信息，为开发者推荐合适的 Web API。

传统推荐方法通常依赖 Mashup 与 Web API 之间的交互信号，但在真实场景中，Mashup 和 Web API 的文本描述中包含了大量功能语义信息。如果只依赖稀疏的交互关系，模型很容易在长尾样本上学习不足；而如果图卷积层数过深，还会进一步引入噪声传播和表示过平滑问题。

论文的出发点是：**能不能让文本语义不只是作为附加特征，而是直接参与图结构构建与表示学习监督，从而反过来改善图协同过滤的质量？**

---

## 核心思想

该工作尝试将**文本语义相似性**引入图协同过滤框架中，使模型不仅学习 Mashup-API 交互关系，还能在表示学习过程中受到文本相似性的显式监督。

整体方法可以概括为四步：

1. **文本表示学习**  
   使用 Sentence-BERT 对 Mashup 和 Web API 的文本描述进行编码，得到语义向量表示。

2. **构建去噪相似图**  
   根据文本表示之间的相似度构建 Mashup 和 API 的相似关系图，只保留达到阈值的高质量连接，从而降低噪声传播的风险。

3. **图协同过滤表示学习**  
   在相似图上采用 LightGCN 进行多层邻居聚合，学习节点的结构表示，并通过多阶邻域信息融合增强表示能力。

4. **文本相似度监督**  
   在图表示学习之外，引入文本相似度监督信号，约束语义相近的节点在表示空间中保持更一致的结构，从而提升节点表示的可区分性与稳定性。

从论文设计上看，TSSGCF 的关键不是简单把文本特征拼接到图模型后面，而是让文本语义同时承担了**建图依据**和**监督信号**两种角色，这也是它区别于一般混合推荐模型的地方。

---

## 实验结果

论文将 TSSGCF 与内容式方法、图式方法和混合式方法进行了系统比较。结果显示，TSSGCF 在所有核心指标上都取得了最优表现：

- Recall：**0.48296**
- NDCG：**0.39449**
- Precision：**0.09768**
- F1 Score：**0.16250**

相较最强基线，论文报告的相对提升分别为：

- Recall：**+9.44%**
- NDCG：**+20.14%**
- Precision：**+8.38%**
- F1 Score：**+8.56%**

论文还设计了三组消融版本：

- `TSSGCF-GCN`：去掉基于 LightGCN 的图表示学习
- `TSSGCF-SBERT`：去掉基于 Sentence-BERT 的文本表示
- `TSSGCF-TSS`：去掉文本相似度监督

消融结果表明，完整模型始终优于这三种删减版本。这说明：

- 去噪相似图能够减少无关连接带来的噪声传播；
- 文本表示本身对捕捉功能语义非常关键；
- 文本相似度监督能够缓解数据不平衡问题，并进一步提升 Recall 和 NDCG 等关键指标。
